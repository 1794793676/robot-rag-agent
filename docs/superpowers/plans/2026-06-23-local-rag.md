# Local Lightweight RAG Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Build a complete local FastAPI + Vue RAG document knowledge-base that runs offline with deterministic fake embeddings and can use DashScope embeddings in production.

**Architecture:** SQLite stores canonical metadata, chunks, and float32 vectors. A rebuildable vector-store adapter selects hnswlib when installed and NumPy cosine search otherwise. FastAPI services coordinate ingestion and retrieval; Vue provides one compact management/testing page.

**Tech Stack:** Python 3, FastAPI, SQLAlchemy, pydantic-settings, NumPy, optional hnswlib, httpx, pypdf, python-docx, Vue 3, Vite, Axios.
